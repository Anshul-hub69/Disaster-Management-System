# DDVRCS – Disaster Volunteer Registration & Coordination System

A web-based Disaster Volunteer Registration & Coordination System developed to streamline volunteer management, disaster response, resource allocation, and SOS request handling. The project is managed using Agile Scrum methodology with Jira for sprint planning and project tracking.

## 📌 Project Overview

The Disaster Volunteer Registration & Coordination System (DDVRCS) helps disaster management teams efficiently manage volunteers, disaster zones, available resources, SOS requests, and mission progress during emergencies.

The project demonstrates the implementation of Agile Scrum practices using Jira, including Epics, User Stories, Sprint Planning, Timeline, and Reports.

---

## ✨ Features

- Volunteer Registration
- Volunteer Dashboard
- Volunteer Assignment & Status Tracking
- Interactive Disaster Map
- Disaster Zone Filters
- Resource Inventory Dashboard
- Resource Stock Status
- Resource Search & Filters
- SOS Request Dashboard
- Volunteer Dispatch System
- SOS Request Resolution
